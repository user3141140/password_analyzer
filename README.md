# Password Security Analyzer
A simple CLI tool to check password strength using the zxcvbn library.

# What's New ? (Latest Update)
- **Engine Upgrade:** Switched to the `zxcvbn` library for much smarter entropy and leak-time estimation compared to the previous version.
- **Improved UX:** Added a cleaner CLI interface with clear options to retry or exit.
- **Code Optimization:** Refactored code structure with better error handling and cleaner logic flow.

# Features
- Analyzes password complexity
- Estimates time to leak
- Simple CLI interface

# How to Use
1. Install requirements: `pip install zxcvbn`
2. Run the script: `python main.py`

*Note: This is a learning project for cybersecurity exploration.*
